Data Security and Compliance

View Data Security Status:
Recent Activity Log: Displays recent security-related actions and events.
Security Recommendations: Provides suggestions for improving security posture.
Manage Encryption:
  • Encrypt Data: Encrypts sensitive patient data using AES-256 encryption.
  • Decrypt Data: Decrypts previously encrypted data for authorized access.
  • Encryption Key Management: Manages keys used for encryption/decryption.
  • Encryption Status: Shows the encryption status of the system.
Backup Management:
  • Schedule Backups: Sets up automated backup schedules for patient data.
  • Manual Backup: Allows for on-demand backups of critical data.
  • Backup History: View history of backups, including timestamps and success/failure status.
  • Restore Data: Restore data from previous backup points if needed.
